# pandas_groupby_operations

Este proyecto proporciona operaciones de groupby en pandas utilizando Jupyter Notebooks.

## Configuración del Entorno

1. Asegúrate de tener Python 3 instalado (versión 3.12.1 o superior).

2. Crea un entorno virtual:
   ```bash
   python3 -m venv env
   ```

3. Activa el entorno virtual:
   ```bash
   source env/bin/activate
   ```

4. Instala las dependencias:
   ```bash
   pip install -r requirements.txt
   ```

5. Para ejecutar Jupyter Notebook:
   ```bash
   jupyter notebook
   ```

## Uso

Una vez configurado el entorno, puedes crear y ejecutar notebooks de Jupyter para realizar operaciones de groupby en pandas.
